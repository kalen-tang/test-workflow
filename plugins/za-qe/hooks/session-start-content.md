
## 🧪 【QE 测试左移工具集】

  ① 快速模式（推荐）  →  /za-qe:qe-quick ./docs/plan.md
    从 KM 开发方案直接生成 API 自动化测试用例

  ② 手工案例生成      →  /za-qe:manual-case ./docs/requirement.md
    生成 PlantUML 流程图 + 测试用例 MindMap

    四阶段流程：规范化 → 需求检查 → 手工案例 → 自动化案例

  ④ 需求文档标准化    →  /za-qe:req-parser ./docs/requirement.docx
    将原始需求文档转为标准化 YAML
