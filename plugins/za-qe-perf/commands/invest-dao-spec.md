# invest-dao-spec

INVEST框架数据访问层完整开发规范。当设计DAO/Mapper层、使用Mybatis-Plus或自研Mybatis框架、编写数据库访问代码时使用。包含SBS项目BaseMapper规范和FTC项目CrudMapper规范。

## 使用方式

在Claude Code中直接调用：

```bash
/invest-dao-spec
```

## 适用场景

- 创建DAO层Mapper接口
- SBS项目继承BaseMapper
- FTC项目继承CrudMapper
- 编写mapper.xml SQL语句
