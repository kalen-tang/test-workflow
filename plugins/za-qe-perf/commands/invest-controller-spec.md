# invest-controller-spec

INVEST框架Controller层完整开发规范。当编写Controller、处理HTTP请求响应、实现RESTful API、处理分页查询时使用。包含ResponseData规范、分页处理、Swagger注解等核心内容。

## 使用方式

在Claude Code中直接调用：

```bash
/invest-controller-spec
```

## 适用场景

- 创建Controller层RESTful API接口
- 配置ResponseData统一响应结构
- 处理分页查询请求
- 编写Swagger API文档注解
