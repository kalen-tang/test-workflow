# za-qe-perf:overview

INVEST微服务框架技术栈和分层架构概览。当需要了解INVEST框架整体架构、技术选型、目录结构时使用。支持银行投资业务系统开发，适用于项目初期架构理解和新人onboarding。

## 使用方式

在Claude Code中直接调用：

```bash
/za-qe-perf:overview
```

## 功能说明

此命令将激活 `invest-framework-overview` 技能，提供以下内容：

- INVEST框架核心技术栈介绍（Spring Boot、MyBatis-Plus等）
- 分层架构设计（Controller-Service-DAO）
- 项目目录结构说明
- 微服务组件和依赖关系
- 适用于新项目启动和团队新人培训

## 适用场景

- 了解INVEST框架Spring Boot + MyBatis-Plus技术栈
- 理解Controller-Service-DAO分层架构
- 银行投资业务系统新人onboarding
- 项目初期架构理解和技术选型
