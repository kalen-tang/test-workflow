# invest-framework-overview

INVEST微服务框架技术栈和分层架构概览。当需要了解INVEST框架整体架构、技术选型、目录结构时使用。支持银行投资业务系统开发，适用于项目初期架构理解和新人onboarding。

## 使用方式

在Claude Code中直接调用：

```bash
/invest-framework-overview
```

## 适用场景

- 了解INVEST框架Spring Boot + MyBatis-Plus技术栈
- 理解Controller-Service-DAO分层架构
- 银行投资业务系统新人onboarding
