# invest-feign-guide

INVEST Feign服务调用指南。用于OpenFeign服务间调用配置和使用。当需要配置Feign客户端、实现服务间调用、处理Feign接口时使用。

## 使用方式

在Claude Code中直接调用：

```bash
/invest-feign-guide
```

## 适用场景

- 配置OpenFeign客户端接口
- 实现微服务间远程调用
- 处理Feign降级和超时配置

## 关键词

invest-feign, open-feign, feign-client, remote-call, service-to-service, feign-fallback, microservice-call, feign-config
