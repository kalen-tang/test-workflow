# invest-database-guide

INVEST数据库ORM使用指南。用于MyBatis-Plus配置、分页查询、TransactionUtils事务处理等数据持久化场景。

## 使用方式

在Claude Code中直接调用：

```bash
/invest-database-guide
```

## 适用场景

- 配置MyBatis-Plus ORM增强功能
- 实现IPage分页查询
- 使用TransactionUtils编程式事务管理数据持久化
