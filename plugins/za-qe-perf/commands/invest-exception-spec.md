# invest-exception-spec

INVEST框架异常处理完整规范。当处理业务异常、设计错误码、实现异常处理机制、统一异常响应时使用。包含BusinessException使用、ResourceHandler接口、错误码枚举设计等内容。

## 使用方式

在Claude Code中直接调用：

```bash
/invest-exception-spec
```

## 适用场景

- 抛出BusinessException业务异常
- 设计错误码枚举类
- 实现ResourceHandler国际化接口
- 统一异常响应处理
