# invest-entity-spec

INVEST框架实体类完整开发规范。当设计数据库实体、配置MyBatis-Plus注解、处理持久化对象时使用。包含实体类设计、字段注解、命名规范等内容。

## 使用方式

在Claude Code中直接调用：

```bash
/invest-entity-spec
```

## 适用场景

- 设计数据库实体类
- 配置@TableName和@TableField注解
- 处理MyBatis-Plus持久化对象映射
